# Meal Planner

Jetpack Compose приложение для планирования блюд и формирования списка покупок.

## Реализовано
- Экран меню со списком блюд и выбором для списка покупок.
- Экран добавления блюда с валидацией ингредиентов.
- Экран списка покупок с суммированием ингредиентов и экспортом в PDF.
- Экран настроек на DataStore.
- Навигация через `NavHost` и нижнее меню.

## Запуск
```bash
./gradlew assembleDebug
```
